================================================================================
âš ï¸ TROUBLESHOOTING
================================================================================

PROBLEM: API key hits monthly limit
SOLUTION: App automatically uses free fallback APIs (Coinbase, Alpha Vantage)

PROBLEM: Port 5000 is busy
SOLUTION: 
1. Kill existing processes: Get-Process | Where-Object {$_.ProcessName -eq "python"} | Stop-Process
2. Or change port in app.py

PROBLEM: Dependencies missing
SOLUTION: Run: pip install -r requirements.txt

PROBLEM: Virtual environment not found
SOLUTION: Create new one: python -m venv .venv

PROBLEM: Permission denied
SOLUTION: Run PowerShell as Administrator
